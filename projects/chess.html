Chess engine

When I first started learning C# around november 2020, the first project I started with was a game of tictactoe. After that I began making a chess game for 2 human players. I just started programming it without really knowing how I would do it. I decided to just use a singe dimensional array. Its not 64 as you would expect, as I changed it to 71 later to put flags inside the array. This way the board state is part of the board when passed around. I'm writing this long after the project so I dont really remember everything about the code.
One thing I remeber being tricky was reversing the board array so the board flipped depening on which players turn.
After the first chess game I read some stuff about chess engines, and found a youtube guide of a chess engine using bitboards written in java. I don't know anything about java, but I managed to follow along, and when I felt like it change stuff how I wanted it to work.
Using 12 bitboards to represent the position of each piece, seemed like a genius idea so thats why I chose this representation.
